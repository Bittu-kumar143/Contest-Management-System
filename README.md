# Contest Management System

A **menu-driven Contest Management System** built using **Object-Oriented Programming (OOP)** principles.

## Key Features
- Manage **contests, participants, organizers, and admins**
- Supports multiple contest types:
  - Coding
  - Chess
  - Debate
- Dynamic **score calculation, leaderboards, and prize allocation**
- **Leaderboard sorting** and **payroll computation**

## Technical Highlights
- OOP concepts (inheritance, polymorphism, virtual functions)
- File handling for persistent data storage
- Template usage for flexible data management

## Purpose
This project demonstrates strong fundamentals in **OOP design, data handling, and system modularity** through a real-world contest management scenario.
